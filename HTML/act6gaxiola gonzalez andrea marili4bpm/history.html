Historia de html

    Linea de tiempo

  • 1980

    • A finales de 1980 Tim Berners propuso el lenguaje html junto con el protocolo http, su finalidad era crear un medio para poder compartir informacion entre grandes fisicos.

  • 1991

    • El primer documento formal con la descripcion de html se publico en 1991 bajo el nombre HTML tags.

  • 1993

    • La propuesta oficial para convertir html en estandar se realizo en 1993 por parte del organismo IETF (internet engineering task force), pero fue fracasada y por otra parte se definieron las etiquetas para imagenes, tablas y formularios.

  • 1995

    • El organismo IETF organiza un grupo de trabajo de html y logra publicar, el 22 de septiembre de ese mismo a√±o, el estandar html 2.0, el primero.

  • 2010

    • Muchos sitios web fuertes como Youtube implementa el uso de html 5.

  • 2013

    • Se suman mas empresas a usar html 5 gracias a los celulares y ademas del uso de sistemas operativos IOS y Android.
